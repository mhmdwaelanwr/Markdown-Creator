## üêõ Reporting Bugs

If you believe you've found a bug, please open an issue on GitHub using our **Bug Report Template**.

1.  Go to the [Issues](../../issues) tab.
2.  Click **New Issue**.
3.  Select **Bug Report**.
4.  Fill out the template with as much detail as possible (steps to reproduce, screenshots, logs).

## üí° Feature Requests

We welcome your ideas! If you have a suggestion for a new feature or improvement:

1.  Go to the [Issues](../../issues) tab.
2.  Click **New Issue**.
3.  Select **Feature Request**.
4.  Describe the feature, why it's useful, and how it should work.
